# Give Up Github!

This account is exclusively for contributions.
GitHub is not a good place to work on Free and Open Source Software.
I strongly discourage using GitHub. Change to an alternative platform,
like [Codeberg](https://codeberg.org), or a community-hosted GitLab instance, like [GitGud](https://gitgud.io). 

https://sfconservancy.org/GiveUpGitHub/
